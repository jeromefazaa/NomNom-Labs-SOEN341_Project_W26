# Backend Express Application

This is a basic Express application structured to demonstrate a simple backend setup.

## Project Structure

```
Backend
├── src
│   ├── app.js
│   ├── controllers
│   │   └── index.js
│   ├── routes
│   │   └── index.js
│   └── middleware
│       └── index.js
├── package.json
└── README.md
```

## Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the Backend directory:
   ```
   cd Backend
   ```

3. Install the dependencies:
   ```
   npm install
   ```

## Usage

To start the application, run:
```
npm start
```

The server will start on the specified port (default is 3000). You can access the application at `http://localhost:3000`.

## API Endpoints

- `GET /`: Returns a welcome message from the IndexController.
